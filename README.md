# 🧠 MindMosaic

### AI-Powered Companion for Dementia & Alzheimer's Care

A mobile healthcare platform that helps **caregivers manage patients** while assisting **early-stage dementia patients** with memory reinforcement, daily schedules, and AI-powered support.
